# Analyse_donnees_metabarcoding_18S_protistes
Les protistes sont des microorganismes eucaryotes unicellulaires.

## mode de déplacement

Les protistes se distinguent en fontion de leur mode de déplacement, on a les ciliés, les pseudopodes, les flagellés, les biflagellés

## mode de nutrition

On distingue les autotrophes (les algues), les hététrotrophes (bactérivotres) et les mixotrophes

## role

Dans le sol, les protistes améliorent l'absorption des minéraux par les champignons mycorhiziens arbusculaires qui vivent en symbiose avec les racines des plantes.
